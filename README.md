## about

this project is a personal aesthetic carrd-inspired website made with pure html & css.

the idea was to create something that feels:
- cinematic
- emotionally distant
- soft but unstable
- like a real person hiding inside the internet

---

## pages

✦ `index.html` → main profile  
✦ `about.html` → personal/about page  
✦ `archive.html` → thoughts & archives  
✦ `letterboxd.html` → favorite films & media
